Практический опыт

Self-hosted сервер на Debian

домашняя инфраструктура · текущий проект

  • Развернул и поддерживаю мультисервисную инфраструктуру: Jellyfin, Nextcloud, Immich, Navidrome, Kiwix, Matrix/Synapse и другие, всего 10+ сервисов.
  • Настроил отказоустойчивое хранилище RAID 1 (mdadm, 2×1 ТБ) для медиа и пользовательских данных.
  • Поднял reverse proxy (Traefik) с автоматическим выпуском SSL-сертификатов Let's Encrypt для всех поддоменов.
  • Развернул Matrix/Synapse с OIDC-аутентификацией через Ansible; самостоятельно диагностировал и устранил конфликты systemd и Docker-сетей при деплое.
  • Развернул multi-node кластер k3s, разобрал и устранил инцидент ImagePullBackOff после миграции сервиса.
  • Настроил прозрачное VPN-туннелирование на уровне сетевого шлюза (sing-box, VLESS).
  • Написал и задеплоил собственный backend API (Node.js + MariaDB).

Это можно потыкать. Интерактивная схема инфраструктуры и разбор инженерных кейсов.
